\# Telegram Advanced Forward Bot



\## What this project does

yeh telegram bot h jo message k filter kr ke jaisa forward krwana ho aur edit krwa kr bhejna ho toh wo sb krta h 



\## Features

\- Multi source channels

\- Multi target channels

\- Keyword based forwarding

\- Duplicate protection

\- Delay system

\- Logs system



\## How to run

python forward\_bot.py



\## Why I built this

yeh maine isliye bnaya h kyuki mujhe aaisa system chahye tha jisse mai telegram ke kaam ko automate kr sku
